# RC-aux Project Page

Static project page for **Predictive but Not Plannable: RC-aux for Latent World Models**.

The site is built with plain HTML, CSS, and JavaScript. GitHub Pages deployment is handled by `.github/workflows/pages.yml`.

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
